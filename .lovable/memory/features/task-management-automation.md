---
name: Task Management Automation
description: Task management features for staff — NO automated task digest email (removed in v3)
type: feature
---
The platform includes a Task Manager for staff to track account-related tasks with due dates, priorities, and assignments. 

**Important**: The automated `task-digest` email template was removed in v3.0. There is NO daily task digest email. The `send-reminder-emails` cron only handles four reminder types: application-not-started, application-reminder, info-request-reminder, and invite-reminder.
